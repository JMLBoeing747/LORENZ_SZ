Enregistrer un message

Vous pouvez décider d'enregistrer des messages après les avoir déchiffrés dans le but d'en garder une copie à des fins d'archivage ou de souvenir. Le message enregistré sera alors inscrit dans l'historique.

Étapes pour enregistrer un message

  1. Commencez par déchiffrer un message.
  2. Lorsque déchiffré, appuyez sur CTRL + S.
  3. LORENZ enregistre le message et vous affiche un message de réussite lorsque terminé.

Consulter un message enregistré

Lorsque vous enregistrez un message, celui-ci est retranscrit dans l'historique des messages. Pour les reconsulter, il suffit d'y accéder et d'atteindre ceux qui vous plaisent. Pour ce faire :

  1. Accédez au menu principal.
  2. Appuyez sur H. Ceci vous fera basculer dans le menu de l'historique.
  3. À l'affichage du menu de l'historique, appuyez sur H une seconde fois.
  4. Recherchez le numéro du message que vous désirez consulter dans la liste. Pour changer de page d'historique, appuyez sur la flèche droite pour avancer, ou sur la flèche gauche pour reculer.
  5. Lorsque trouvé, tapez ce numéro et appuyez sur ENTRÉE.

Supprimer un message enregistré

Pour supprimer un message enregistré dans l'historique :

  1. Accédez au menu principal.
  2. Appuyez sur H. Ceci vous fera basculer dans le menu de l'historique.
  3. À l'affichage du menu de l'historique, appuyez sur H une seconde fois.
  4. Recherchez le numéro du message que vous désirez supprimer dans la liste. Pour changer de page d'historique, appuyez sur la flèche droite pour avancer, ou sur la flèche gauche pour reculer.
  5. Lorsque trouvé, appuyez sur DELETE et tapez le numéro du message. La sortie écrite aura l'air de ceci : Del : <index>.
  6. Appuyez sur ENTRÉE pour valider et sur X pour confirmer.

Pour supprimer un message enregistré d'une catégorie :

  1. Accédez au menu principal.
  2. Appuyez sur H. Ceci vous fera basculer dans le menu de l'historique.
  3. À l'affichage du menu de l'historique, appuyez sur C.
  4. Inscrivez le numéro de la catégorie où le message à supprimer se trouve et appuyez sur ENTRÉE.
  5. Recherchez le numéro du message que vous désirez supprimer dans la liste. Pour changer de page d'historique, appuyez sur la flèche droite pour avancer, ou sur la flèche gauche pour reculer.
  6. Lorsque trouvé, appuyez sur DELETE et tapez le numéro du message. La sortie écrite aura l'air de ceci : Del : <index>.
  7. Appuyez sur ENTRÉE pour valider et sur X pour confirmer.

Catégoriser un message

Si votre historique devient trop gros et que vous vous perdez à travers tous ces artéfacts de discussion, vous pouvez répartir les messages enregistrés en catégories.

Pour créer une nouvelle catégorie :

  1. Accédez au menu principal.
  2. Appuyez sur H. Ceci vous fera basculer dans le menu de l'historique.
  3. À l'affichage du menu de l'historique, appuyez sur N pour accéder à l'interface de création de catégorie.
  4. Entrez un nom pour votre nouvelle catégorie et validez avec ENTRÉE.

Par la suite, pour ajouter un message dans la nouvelle catégorie :

  1. Consultez le message que vous désirez catégoriser (voir les étapes pour Consulter un message enregistré ci-haut)
  2. À l'affichage de sa description, appuyez sur C.
  3. Inscrivez l'index de liste correspondant à la catégorie dans laquelle le message devra être placé.
