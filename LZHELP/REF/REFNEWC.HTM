N : Nouvelle catégorie

Référence technique sur l'interface de création de nouvelle catégorie de l'historique.

Présentation

Cette commande permet la création d'une nouvelle catégorie destinée à classifier les messages de l'historique. Le message d'entrée est le suivant :


	Pour annuler, appuyez sur ESC, ou sur ENTRÉE sans rien écrire.

	Nom de la nouvelle catégorie :

Entrées

Une chaîne de caractères représentant le nom de la nouvelle catègorie à créer. Ceci ajoutera une nouvelle entrée dans le fichier CATEGORY.LZI, ou le laissera inchangé si l'entrée vaut "\n" (caractère de retour à la ligne 0xA

Touches d'échappement

  • ENTRÉE (Retour à la ligne)
  • ESC (Touche Escape)

Messages de sortie possibles

Catégorie <nouvelle catégorie> créée avec succès !
Succès : Signifie que la catégorie du nouveau nom a été créée avec succès et inscrite dans le fichier CATEGORY.LZI.
