Paramètres de configuration

Référence technique sur les paramètres de configuration de LORENZ

Présentation

Les paramètres de configuration permettent de personnaliser l'interface de l'application selon les besoins de chaque utilisateur.
Ces données sont sauvegardés sous le fichier LORENZ.INI, situé dans le répertoire %LORENZPATH%\LZPARAMS.

Liste des paramètres

Section Profile

SHOWSENDER
Type : Booléen
Valeur par défaut : False
Interprétation : Si True, permet d'afficher le pseudonyme de l'expéditeur d'un message déchiffré, dans le mode Déchiffrement. Si False, ne l'affiche pas.
PSEUDONAME
Type : Chaîne de caractères (String)
Valeur par défaut : Nom d'utilisateur du compte système associé à l'application, équivalent à la variable d'environnement système %USERNAME%.
Interprétation : Permet d'identifier les messages qui sont expédiés par cet utilisateur auprès des autres utilisateurs LORENZ.

Section Settings

CIPHERFILEDIR
Type : Chaîne de caractères (String)
Valeur par défaut : NULL.
Interprétation : Localise le répertoire de stockage des fichiers de chiffrements longs. Ce répertoire est utilisé pour sauvegarder les chiffrements longs, ou pour les localiser lors de leur déchiffrement.
TRANSTABLE
Type : Entier à 4 chiffres (Integer)
Valeur par défaut : 1234.
Interprétation : Contient la racine de la table des transcriptions, utilisée pour générer la première passe de chiffrement de LORENZ. Un changement de valeur affecte directement l'allure finale des chiffrements générés.
SECRETABLE
Type : Chaîne de caractères (String)
Valeur par défaut : S8H2ALDVFP
Interprétation : Contient la disposition de la table secrète, utilisée pour générer la deuxième passe de chiffrement de LORENZ. Un changement de valeur affecte directement l'allure finale des chiffrements générés

Section Compression

ACTIVCMPRS
Type : Booléen
Valeur par défaut : True
Interprétation : Si True, active le mode de compression par associations. Si False, désactive le mode de compression par associations. Il est à noter que le mode de compression par répétitions est toujours actif, peu importe la valeur de ce paramètre.
CMPRSRATIO
Type : Nombre flottant double-précision (Double)
Valeur par défaut : 0,15
Interprétation : Détermine le taux minimum de compression exigé par l'utilisateur pour l'appliquer sur le chiffrement.
