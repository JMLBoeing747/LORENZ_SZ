Paramètres de configuration

Référence technique sur les paramètres de configuration de LORENZ

Présentation

Les paramètres de configuration permettent de personnaliser l'interface de l'application selon les besoins de chaque utilisateur.
Ces données sont sauvegardés sous le fichier PARAMS.INI, situé dans le répertoire %LORENZPATH%\LZPARAMS.

Liste des paramètres

SHOWSENDER
Type : Booléen
Valeur par défaut : False
Interprétation : Si True, permet d'afficher le pseudonyme de l'expéditeur d'un message déchiffré, dans le mode Déchiffrement. Si False, ne l'affiche pas.
PSEUDONAME
Type : Chaine de caractère (String)
Valeur par défaut : Nom d'utilisateur du compte système associé au programme, équivalent à la variable d'environnement système %USERNAME%. Dans ce cas, le paramètre n'est pas inscrit dans PARAMS.INI.
Interprétation : Permet d'identifier les messages qui sont expédiés par cet utilisateur auprès des autres utilisateurs LORENZ.
CIPHERFILEDIR
Type : Chaine de caractère (String)
Valeur par défaut : NULL. Dans ce cas, le paramètre n'est pas inscrit dans PARAMS.INI.
Interprétation : Localise le répertoire de stockage des fichiers de chiffrements longs. Ce répertoire est utilisé pour sauvegarder les chiffrements longs, ou pour les localiser lors de leur déchiffrement.
