1 : Chiffrement

Référence technique sur le mode Chiffrement de LORENZ

Présentation

Mode permettant le chiffrement d'un message d'encodage ANSI (Windows-1252). Le message d'entrée est le suivant :


	AVIS : Vous pouvez écrire sur plusieurs paragraphes !
	Pour annuler, appuyez sur ESC.
	Pour terminer le message, appuyez sur CTRL + D.
	Entrez le texte à chiffrer :

Entrée

Le message d'encodage ANSI à chiffrer
Peut contenir des commandes attributives.

Sortie

Le message d'entrée chiffré par l'algorithme de chiffrement LORENZ
Le message chiffré est composé de la somme de contrôle, de la clé de chiffrement, de l'attribut SENDER et du message chiffré, comprenant les commandes attributives spécifiées, s'il y a lieu.

Touches d'échappement

  • ESC (Touche Escape)
  • CTRL + D (^D, lorsque déjà écrit)

Messages de sortie possibles

Aucun texte détecté.
Erreur spécifique : N'arrive que lorsque l'utilisateur entre un message contenant uniquement des espaces, le qualifiant ainsi de texte vide.
Le caractère X n'est pas supporté. Néanmoins, il a été remplacé par "?" pour compléter le message.
Avertissement : Signifie que le caractère X n'est pas supporté par le jeu de caractères ANSI et a donc été remplacé par le point d'interrogation (? : 0x3F = 63) afin de combler le message dans le but de ne pas le saccader de lettres absentes, et ainsi d'avertir le récepteur de la présence de lettres manquantes à cet endroit.
