Aspects techniques supplémentaires

Référence technique sur des aspects techniques supplémentaires aux précédentes références.

Présentation

Voici d'autres informations techniques non répertoriées.

Il s'agit du dossier qui contient l'exécutable de l'application LORENZ, le répertoire des paramètres LZPARAMS ainsi que ses extensions importantes (fichiers dll). Normalement situé au chemin d'accès C:\Users\<Nom utilisateur>\AppData\Local\Programs\LORENZSZ, il est accessible via la variable d'environnement %LORENZPATH%.

<Nom utilisateur> correspond au nom utilisateur du compte associé au système d'exploitation.

Il s'agit du dossier contenant les fichiers de données de l'application. Consultez la page de référence Fichiers de données pour accéder à la référence technique de ces fichiers.

Ces fichiers ont été générés lors de la compilation du code source et sont nécessaires pour assurer le bon fonctionnement de l'application.
Ils s'agissent de Dynamic Link Libraries et ils sont au nombre de 4 :

  • clrcompression.dll
  • clrjit.dll
  • coreclr.dll
  • mscordaccore.dll
