2 : Déchiffrement

Référence technique sur le mode Déchiffrement de LORENZ

Présentation

Mode permettant le déchiffrement d'un message préalablement chiffré avec l'algorithme de chiffrement LORENZ. Le message d'entrée est le suivant :


	AVIS : Le texte chiffré peut s'étendre sur plusieurs lignes !
	Validez la dernière ligne en cliquant ENTER sans rien écrire dans cette dernière.
	Entrez le texte à déchiffrer :
	Pour annuler, cliquez ENTER sans rien écrire.

Entrée

Le message préalablement chiffré par l'algorithme de chiffrement LORENZ

Sortie

Le message d'entrée déchiffré. Il s'agit d'un message d'encodage ANSI (Windows-1252).
Si SHOWSENDER est True, le message déchiffré est précédé du nom du pseudonyme de son expéditeur.

Touches d'échappement

  • ENTER (Retour à la ligne)

Messages de sortie possibles

LORENZ WARNING MSG: THIS MESSAGE IS PRIVATELY DESTINATED TO ANOTHER USER
Alerte : Signifie que l'opération de déchiffrement a détecté l'attribut PRIVATE et a constaté que la valeur de l'attribut (soit le pseudonyme à qui s'addresse le message en privé) n'est pas identique à celui du récepteur réel.
Ce message n'est pas valide.
Erreur générique : Signifie que le message entré n'a pas été chiffré par l'algorithme de chiffrement LORENZ ou qu'il l'a mal été. Il s'agit d'une erreur générique et est donc affichée à la fin de toutes les erreurs spécifiques.
ERREUR : Un ou des caractères sont erronés
Erreur spécifique : Signifie que certains caractères ont été détectés comme intrus ou inattendus par LORENZ et qu'il s'agit d'une erreur de transcription du chiffrement.
ATTENTION LE MESSAGE PEUT CONTENIR DES DÉFORMATIONS DUE À UNE MODIFICATION ACCIDENTELLE OU MALINTENTIONNÉE DE VALEURS DANS LE CHIFFREMENT
Alerte : Signifie que l'opération de déchiffrement a décelé des déformations au sein du message chiffré. Cela est soulevé lorsque LORENZ n'arrive pas à trouver une certaine transcription de la table de chiffrement générée, ou lorsque la somme de contrôle calculée est différente de celle reçue.
ERREUR FATALE CODE 55 : Le chiffrement n'est pas authentique.
Erreur spécifique : Signifie que la clé de chiffrement s'est avérée invalide puisque la génération de la table de chiffrement a échoué.
