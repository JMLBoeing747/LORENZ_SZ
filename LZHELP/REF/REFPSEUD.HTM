P : Modifier le pseudo

Référence technique sur la commande Modifier le pseudo de LORENZ

Présentation

Commande permettant d'accéder à l'interface de modification du pseudonyme utilisateur. Le message d'entrée est le suivant :


	Le pseudo est le nom qui sera affiché à tous les récepteurs pouvant déchiffrer le message
	et qui ont activé l'affichage de l'expéditeur. Si vous désirez le modifier, tapez
	ci-dessous le nouveau pseudo à utiliser pour les chiffrements futurs. Si vous désirez le
	réinitialiser, taper tel quel votre nom d'utilisateur système ou $DEFAULT (la casse
	n'importe pas). Sinon, appuyer sur ENTRÉE sans rien écrire.
	Actuel : <Pseudo actuel>
	Nouveau >>>

N.B. : <Pseudo actuel> correspond à la valeur de l'UID de l'utilisateur.

Entrée

Le nouveau pseudonyme à utiliser. Ceci affectera la nouvelle valeur au paramètre PSEUDONAME, ou le laissera inchangé si l'entrée vaut "\n" (caractère de retour à la ligne 0xA).

Touches d'échappement

  • ENTRÉE (Retour à la ligne)
  • ESC (Touche Escape)

Messages de sortie possibles

Nouveau pseudo enregistré !
Succès : Signifie que la valeur entrée a été nouvellement affectée au paramètre PSEUDONAME avec succès. Le paramètre PSEUDONAME est alors immédiatement inscrit dans le fichier LORENZ.INI.
Pseudo identique au précédent. Aucun changement à faire.
Avertissement : Signifie que la valeur entrée correspond exactement au pseudonyme attribué avant l'opération et ne procédera donc à aucun changement ni à aucune réécriture du paramètre PSEUDONAME.
Valeur par défaut réinitialisée
Avertissement : Signifie que la valeur entrée correspond exactement au pseudonyme par défaut (Voir paramètres de configuration). Le paramètre PSEUDONAME ne sera pas inscrit dans le fichier LORENZ,INI.
