Fichiers de données

Bienvenue à la page principale consacrée aux fichiers de données utilisées par LORENZ.

L'application n'utilise que deux types de fichiers de données, tous placés dans le dossier LZPARAMS du répertoire principal de LORENZ :

  • Fichiers LZI
  • Fichier LORENZ.INI
  • Fichier PARAMS.INI (obsolète)

Il est très important de ne pas les modifier, voire même de les ouvrir, afin de ne pas corrompre les données de ces fichiers, sauf si vous savez ce que vous faites.
