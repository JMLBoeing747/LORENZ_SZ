Menu principal

Référence technique sur les commandes du menu principal

Présentation

Le menu principal dispose de 11 commandes, dont les neuf d'instruction :


	1 : Chiffrement
	2 : Déchiffrement
	L : LID
	J : Jeu
	H : Historique

	[ ] Afficher l'expéditeur (S)
	[ ] Activer la compression (C)

	P : Modifier le pseudo
	R : Modifier/Spécifier le répertoire des chiffrements
	O : Options de chiffrement

	F1 : AIDE

Et la commande d'échappement, la touche ESC (touche d'échappement).

Descriptif des commandes

1 : Chiffrement
Commande : La touche 1 du clavier standard ou du pavé numérique.
Fonction : Permet d'accéder au mode de chiffrement de LORENZ.
2 : Déchiffrement
Commande : La touche 2 du clavier standard ou du pavé numérique.
Fonction : Permet d'accéder au mode de déchiffrement de LORENZ.
L : LID
Commande : La touche L du clavier standard.
Fonction : Permet d'accéder à l'interface du LID pour permettre à l'utilisateur d'identifier le sien.
J : JEU
Commande : La touche J du clavier standard.
Fonction : Permet d'accéder à l'interface de jeu de LORENZ.
H : HISTORIQUE
Commande : La touche H du clavier standard.
Fonction : Permet d'accéder au menu de l'historique.
Afficher l'expéditeur (S)
Commande : La touche S du clavier standard.
Fonction : Interrupteur inversant la valeur du paramètre SHOWSENDER.
Activer la compression (C)
Commande : La touche C du clavier standard.
Fonction : Interrupteur inversant la valeur du paramètre ACTIVCMPRS.
P : Modifier le pseudo
Commande : La touche P du clavier standard.
Fonction : Permet d'accéder à l'interface de modification du pseudonyme utilisateur.
F1 : AIDE
Commande : La touche de fonction F1 du clavier standard.
Fonction : Permet d'ouvrir le fichier d'aide compilé HTML du programme, LZHELP.CHM.
