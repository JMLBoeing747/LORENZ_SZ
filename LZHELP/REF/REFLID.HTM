L : LID

Référence technique sur l'interface LID

Présentation

Interface permettant d'afficher le LORENZ IDentifier correspondant au code de six caractères unique à chaque utilisateur LORENZ.
Cette interface permet ainsi d'identifier le code de l'utilisateur à des fins de consultation, notamment lorsqu'il s'agit de planifier une communication privée. L'interface correspond à ceci :


	Le LID vous permet de vous identifier de façon unique envers tous les utilisateurs LORENZ.
	Vous avez besoin de connaître le LID de votre correspondant si vous désirez lui envoyer un
	message en privé. Le LID ne peut pas être modifié car il est propre à chaque clé de produit.
	Voici votre LID de quatre caractères :

	>>> <LID> <<<

N.B. : <LID> correspond à la valeur du LID de l'utilisateur.

Touches d'échappement

  • Toutes les touches du clavier (à l'exception des touches de contrôle)
