Chiffrer un message

Dans cette page d'aide, vous trouverez toutes les informations utiles à savoir pour chiffrer un message avec LORENZ.

Étapes pour chiffrer un message sans commandes spéciales

  1. Dans le menu principal, choisir l'option 1 : Chiffrement en appuyant sur la touche 1 du clavier ou du pavé numérique
  2. Lorsque l'instruction fera son apparition, entrez le message à chiffrer. Pour terminer, appuyez sur CTRL + D (^D).
  3. LORENZ chiffre le message et vous l'affiche.
    1. Si vous avez Windows 10 ou ultérieur, vous pourrez directement sélectionner le texte chiffré comme dans un éditeur de texte normal et copier avec Ctrl + C.
    2. Sinon, cliquez avec le bouton droit de votre souris sur la barre d'identification de fenêtre (Là où est inscrit LORENZ SCHLÜSSELZUSATZ). Dans le menu contextuel qui apparaitra, choisissez Modifier > Sélectionner. Sélectionnez le texte en prenant soin d'englober tout le message dans le rectangle de sélection. Enfin, cliquez avec le bouton droit la barre d'identification et, dans le menu, choisissez Modifier > Copier.

Chiffrer un message avec des commandes attributives

Si vous avez besoin de spécifier des commandes attributives au chiffrement, tel que PRIV, vous n'avez simplement qu'à l'ajouter à l'écrit dans votre message.

  • Pour consulter l'aide rapide sur PRIV, consultez la page Écrire en privé sous la rubrique Commandes supplémentaires.

Pour la référence complète sur les commandes attributives, référez-vous à la page Attributs sous la rubrique Référence technique.

Si on vous dit que le chiffrement contient plus de 4094 caractères...

Un chiffrement de plus de 4094 caractères ne peut pas être collé dans le mode de déchiffrement LORENZ. Pour contourner ce problème, LORENZ crée un fichier à part et y inscrit le chiffrement intégral. Pour que cela puisse se faire, vous devez simplement spécifier son nom. LORENZ vous affiche alors le chemin d'accès complet au fichier, tout juste après l'avertissement, pour vous aider à le retrouver.

Ce fichier sera placé dans le répertoire des chiffrements que vous avez préalablement spécifié en options. Si aucun répertoire n'a été spécifié, LORENZ vous le demandera tout juste après l'inscription du nom du fichier.
